# Theme Comparison

This document demonstrates both **dark** and **light** themes.

## Dark Theme (Default)
- Background: `#0d1117` (GitHub dark)
- Text: `#e6edf3` (light gray)
- Accent: `#58a6ff` (blue)

## Light Theme (Available)
- Background: `#ffffff` (white)
- Text: `#24292f` (dark gray)
- Accent: `#0969da` (blue)

## Code Example

```go
func main() {
    fmt.Println("Theme comparison")
}
```

## Table Example

| Feature | Dark | Light |
|---------|------|-------|
| Background | Dark gray | White |
| Text | Light gray | Dark gray |
| Syntax | Monokai | GitHub Light |

> Both themes use the same clean layout and typography.
